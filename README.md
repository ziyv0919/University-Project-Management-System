# 高校课题管理系统

## 项目概述
本项目旨在解决高校课题管理流程不规范、效率低的问题，实现课题申报、申请、进度跟踪等全流程数字化。系统面向管理员、教师、学生三类角色，提供课题全生命周期管理功能，提升高校科研管理效率。

## 核心功能
1. **用户管理**：支持注册、登录、个人信息维护、密码修改，基于角色的权限控制（管理员/教师/学生）。
2. **课题申报与审核**：教师可发布课题并上传参考资料，管理员审核通过后课题对学生开放。
3. **课题申请与审批**：学生浏览课题并提交申请，教师审批并给出通过/拒绝结果，系统实时通知。
4. **进度汇报与评价**：学生分阶段提交进展报告（含附件），教师填写评语和评分。
5. **通知与公告**：系统自动发送审核结果通知，管理员可发布面向全体或特定角色的公告。
